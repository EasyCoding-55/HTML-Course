Mozilla Developer Network (MDN)

MDN Web Docs formerly known as Mozilla Developer Network is an open-source, collaborative project documenting Web platform technologies, including HTML, CSS, JavaScript, and Web APIs. They also provide an extensive set of learning resources for beginning developers and students, since 2005.

The details and summary elements.

The details element generates a revealing widget, which only displays information when the widget is toggled to the "open" position. The summary element must contain a summary or label.

This is an example of aside element.

The aside element represents a section of a page that contains content that is slightly related to but different from the content surrounding the aside element. For example, a key phrase, quotation, advertising, or sidebars.

The widget is usually represented by a small triangle that rotates to indicate open or closed positions, along with a phrase next to the triangle. The contents of the summary element are used as the label for the revealing widget.
